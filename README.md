# Urban Cleaning Services Website

Official website for **Urban Cleaning Services**, a professional residential and commercial cleaning company based in Nairobi, Kenya.

## 🚀 About The Project

This is a modern, responsive website built to:

- Showcase cleaning services
- Display pricing
- Allow customers to request services
- Enable direct WhatsApp booking
- Build trust and online presence

The site is optimized for speed, mobile users, and local SEO.

---

## 🛠️ Built With

- Next.js 16 (App Router)
- React
- TypeScript
- Tailwind CSS
- Lucide Icons
- React Icons

---

## 📌 Features

- Responsive modern design
- Service listing with pricing
- WhatsApp floating chat button
- Contact page with booking form
- Online status indicator
- Optimized service images
- Clean, professional UI
